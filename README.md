# usenix26-upcycling-repo
The files in this repository are for the USENIX 2026 paper: "Investigating Educational Collaborations with Upcycled Computing Equipment on Student-Run Cybersecurity Clubs"

- `interview_questions_NPP.txt` contains the list of interview questions for NPP affiliates described in Section 3.2.
- `interview_questions_non_NPP.txt` contains the list of interview questions for cyber club students & alumni described in Section 3.2.
